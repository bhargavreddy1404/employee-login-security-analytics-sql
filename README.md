# SQL-Based Employee Login Monitoring and Security Analytics System

## Project Overview

This project is a MySQL-based Employee Login Monitoring and Security Analytics System designed to track employee login activities and generate security-related reports.

The system stores employee details, department information, and login records. SQL queries are used to analyze login behavior, identify failed login attempts, generate departmental statistics, and monitor user activity.

## Objectives

* Monitor employee login activity
* Track successful and failed login attempts
* Generate department-wise reports
* Perform security analytics using SQL
* Demonstrate database design and SQL querying skills

## Technologies Used

* MySQL
* SQL
* GitHub

## Database Tables

### Departments

Stores department information.

### Users

Stores employee details.

### Login_Logs

Stores login activity records.

## Features
